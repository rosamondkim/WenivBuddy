# 📄 WenivBuddy – 요구사항 명세서 (SRS)

부트캠프 Q&A 분석 및 추천 AI 도우미

---

## 1. 프로젝트 개요

**WenivBuddy**는 부트캠프 운영/강의 환경에서 반복적으로 등장하는 학생 질문을 자동 분석하여

- 과거 답변
- 유사 질문
- 캡처 이미지 포함된 답변
- 추가 학습 자료(Reference)
- FAQ 자동 등록 추천

까지 제공하는 **AI 기반 Q&A 어시스턴트**입니다.

Frontend / Backend / 데이터분석 / 디자인 등 다양한 주제가 혼재한 캠프 환경에서  
**정확하고 신뢰도 있는 답변을 빠르게 제공**하는 것을 목표로 합니다.

---

## 2. 주요 사용자

### 강사 (Instructor)

- 수업 중 들어오는 질문을 즉시 해결
- 반복되는 질문을 빠르게 정리 및 응답

### 멘토 / 운영자 (TA / Manager)

- 반복 질문 관리
- FAQ 자동 축적
- 교육 운영 효율 향상

## 4. 기능 요구사항 상세

---

### 4.1 질문 입력 기능

#### 기능 요구사항

1. 사용자는 질문을 텍스트로 붙여넣을 수 있다.
2. 입력 전 **관련 분야(Category)** 체크박스를 제공한다.

   **선택 옵션 예시:**

   - Frontend (HTML/CSS/JS/React)
   - Backend (Node, Spring)
   - Data (Python/Pandas/SQL)
   - Git/GitHub
   - Tools (VSC, Figma 등)

3. 질문은 **최소 5자 이상** 입력해야 검색 가능하다.

#### 목적

- 분야에 따른 답변 정확도 향상
- 다양한 트랙이 섞여 있는 캠프 환경에 최적화

---

### 4.2 이전에 제공했던 답변 조회

#### 기능 요구사항

- 내부 Q&A 저장소(JSON/DB)에서  
  **동일 카테고리 + 의미적으로 유사한 질문**을 검색한다.
- 검색된 결과는 아래 정보를 포함한다:
  - 답변 텍스트
  - 답변 작성 시각
  - 답변 작성자(강사명, 선택)
  - **캡처 이미지 썸네일 or 이미지 경로**
- 최대 3개까지 노출한다.

#### 목적

- 과거 신뢰도 높은 답변을 빠르게 참고
- VSC/Git처럼 이미지 기반 설명이 필요한 질문 처리 용이

---

### 4.3 유사 질문 추천 기능

#### 기능 요구사항

- 입력 질문과 의미적으로 유사한 질문을 **3~5개 제공**
- 과거 로그가 없더라도 **모델이 추론 기반으로 생성 가능**
- 리스트 클릭 시 **질문 + 답변 상세 보기** 팝업 또는 섹션 표시

#### 목적

- 반복 질문 감소
- 학습 확장성 증가 (학생이 스스로 검색 가능)

---

### 4.4 추가 자료(Reference) 추천

#### 기능 요구사항

AI는 질문을 기반으로 자동으로 다음 자료를 추천한다:

- MDN, React Docs, Pandas Docs 등 공식 문서
- CodeSandbox/StackBlitz 예제 코드
- 블로그 튜토리얼
- 디자인/컴포넌트 참고 자료(Figma · UI Kit 등)

각 링크는 **추천 이유(짧은 설명)**를 포함한다.

#### 목적

- 학생이 스스로 탐색하는 학습 능력 향상
- 강사의 자료 수집 부담 감소

---

### 4.5 학생에게 바로 보낼 수 있는 응답 형식 제공

#### 기능 요구사항

- **“복사하기(Clipboard Copy)” 버튼 제공**

- 디스코드/슬랙에 바로 붙여넣기 가능
- 학생 수준에 따른 맞춤형 응답 제공

---

### 4.6 FAQ 자동 등록 추천 기능

#### 기능 요구사항

- 특정 질문이 **3회 이상 반복**되면 시스템이 감지한다.
- 다음 팝업 또는 카드가 나타난다:

> “이 질문은 반복적으로 등장하고 있습니다. FAQ로 등록하시겠습니까?”

- “등록하기” 클릭 시 다음 정보가 FAQ DB에 저장됨:
  - 질문 텍스트
  - 대표 답변
  - 태그/카테고리
  - 참고 이미지(있다면)

#### 목적

- 자동화된 FAQ 구성
- 운영자의 반복 문서 관리 부담 감소

## 5. UI 요구사항

---

### 5.1 상단 영역

- WenivBuddy 로고 또는 텍스트
- 카테고리 필터(체크박스/드롭다운)
- 질문 입력 textarea
- “검색하기 / 분석하기” CTA 버튼

---

### 5.2 결과 영역 (3개 섹션)

“검색하기 / 분석하기” CTA 버튼을 누르면 하단에 결과 영역이 생성된다.

#### ▷ 1. 이전 답변 카드

- 답변 요약 (더보기 버튼)
- 전체 답변
- 캡처 이미지(썸네일)

#### ▷ 2. 유사 질문 카드

- 질문 리스트
- 클릭 시 상세 보기

#### ▷ 3. 추가 레퍼런스 카드

- 공식 문서 링크
- 예제 코드 링크
- 블로그/튜토리얼
- 추천 이유 포함

---

### 5.3 응답 포맷 옵션 영역

- “복사하기” 버튼

---

### 5.4 FAQ 추천 영역

- 반복 질문 감지 시 팝업 또는 카드 표시
- “FAQ로 등록하기” 버튼

---

## 6. 비기능 요구사항

### 성능

- API 요청 후 결과 출력: **1~3초 이내**
- 이미지 로딩은 지연 가능, 텍스트는 우선 렌더링

### 안정성

- 이미지 경로 실패 시 기본 이미지 또는 ALT 표시
- 질문이 비어 있을 경우 검색 버튼 비활성화

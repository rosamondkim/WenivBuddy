   질문의 제목이 'PS C:\Users\user\Desktop\바이브코딩> npm install -g @anthropic-ai/claude-code' 형식처럼 불필요한 드라이브 위치 등의 정보를 제목으로 보여주고 있어. 이 경우 사용자는 내가 원하는 정보인지 제목만 보고 판단하기 힘들어

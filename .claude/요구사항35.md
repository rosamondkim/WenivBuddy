# 질문

이 프로젝트를 기반으로 발표 슬라이드를 만들거야.

만들게 된 동기: 스쿨을 시작하면 늘 커리큘럼이 같기 때문에 질문도 거의 동일한 편이에요.
그러나 매번 멘토가 같진 않죠. 늘 바뀝니다.
근데 저희는 답을 알고있어도 이걸 또 다시 순서대로 입력하고, 캡쳐도 해야되는 번거로움이 있습니다. 몇번이나 다시 검색을 해야해요.
그리고 가끔 바이브코딩 부트캠프에 비개발자분들이 멘토로 들어가면
node나 클로드코드 세팅 과정에서 엄청난 당황스러움을 느끼시게 됩니다.
그래서 부트캠프 질문 모음 사이트를 만들었습니다.

슬라이드 구성과 기능 소개를 하단 답변에 정리해줘

### 슬라이드 4: 핵심 기능 #1 - 스마트 검색

**AI 기반 하이브리드 검색 시스템**

🔍 **검색 방식**

- 텍스트 검색: "윈도우 cmd" → 명령 프롬프트 관련 답변
- 이미지 검색: 에러 스크린샷 업로드 → OCR로 자동 인식
- 자연어 검색: "npm 설치가 안돼요" → 관련 Q&A 자동 매칭

📊 **검색 엔진**

- 로컬 키워드 추출 + LLM 보강
- 신뢰도 기반 자동 선택
- 비용 최적화 ($0.0001/request)

---

### 슬라이드 5: 핵심 기능 #2 - OCR 에러 인식

**스크린샷만 올려도 자동으로 에러 분석**

📸 **작동 과정**

1. 터미널 에러 스크린샷 업로드
2. GPT-4o-mini로 텍스트 자동 추출
3. 프롬프트/경로 자동 제거 (예: `PS C:\Users\...>`)
4. 핵심 에러만 추출 (예: `PSSecurityException`)
5. 관련 해결책 자동 매칭

✨ **정제 기술**

- 터미널 프롬프트 필터링
- 에러 키워드 우선 추출
- 36% 텍스트 압축률

**최신 기술로 구현된 시스템**
